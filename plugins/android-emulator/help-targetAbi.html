Should be the name of the ABI / system image to be used, e.g "armeabi" or "x86".
Additionally a tag can be specified via <Tag>/<ABI>, e.g "google_apis/x86_64".
SDK Tools ≤ 25.2
If empty the default ABI for the select platform will be used. You only need to define this field if you have more than one system image per Android platform installed.
SDK Tools ≥ 25.3
If AVD is created via this plugin, the value is required to run 'avdmanager create avd'.
